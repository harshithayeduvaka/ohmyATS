# 🤖 oh my ATS

A no-fuss ATS compatibility checker built to help job seekers figure out if their resume will survive the bots.

Live site → (https://ohmyats.lovable.app)

---

## What it does

Paste your resume, get a compatibility score. That's it.

Most resumes don't fail because the candidate is underqualified — they fail because the formatting confuses an Applicant Tracking System before a human ever sees them. `oh my ATS` helps you catch those issues before you hit submit.

---

## Features

- ✅ ATS compatibility scoring
- 📄 Resume text analysis
- 🚩 Flags common ATS red flags (tables, columns, weird fonts, missing keywords, etc.)
- ⚡ Fast, browser-based — no sign-up required

---

## Tech Stack

Built entirely with [Lovable](https://lovable.dev) — vibe coded from scratch, no backend required.

- React (via Lovable)
- Tailwind CSS
- Deployed on Lovable's hosting

---

## Why I built this

Job hunting is already exhausting. The last thing anyone needs is to spend hours tailoring a resume only to have it filtered out by a parser that can't read two-column layouts.

I wanted something quick, honest, and actually useful — so I built it.

---

## Run locally

Clone the repo and install dependencies:

```bash
git clone https://github.com/YOUR_USERNAME/ohmyats.git
cd ohmyats
npm install
npm run dev
```

> Requires Node.js 18+
